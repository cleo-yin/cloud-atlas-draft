通过IP网络共享USB设备可以实现一种比较巧妙的远程服务器跨网络使用共享的USB设备，适合测试环境切换USB设备进行服务器测试。

arch linux网站提供了一个 [USB/IP](https://wiki.archlinux.org/index.php/USB/IP) 文档，指导如何实现跨网络共享USB设备，后续有环境可以模拟测试一下，例如跨网络使用U盘，USB key等等。
